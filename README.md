Setting up environment

1- install ganache
2- create a workspace
3- check port
4- git clone https://github.com/fuujazz/Fund-Me-and-Earn-NFT.git
5- cd
6- npm install
7- truffle migrate:ganache
8- cd src
9- npm install
10- install metamask
11- add ganache accounts to metamask
12- npm start
